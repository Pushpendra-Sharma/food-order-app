# food-app

A food ordering app.

## Tech Stack

**Client:** React, Redux Toolkit, TypeScript, TailwindCSS

**Hosted:** Netlify

## Features

- Shimmer UI
- Responsive design
- Restaurants listing page
- Menu page
- Cart page

## Run/Edit Locally

Clone the project

```bash
  git clone https://github.com/Pushpendra-Sharma/food-order-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
